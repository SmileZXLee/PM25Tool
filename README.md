# PM25Tool
### [数据来源](http://www.pm25.com)
### 起因：女朋友是统计学的，导师需要批量收集http://www.pm25.com 的各项数据并统计、整理成Excel，收集过程枯燥乏味无趣，便想着花一点时间写几行代码，减轻负担~
***
### 使用方法
#### 安装必要的包
```
pip install request
```
```
pip install demjson
```
```
pip install xlwt
```
```
pip install pyExcelerator
```
```
pip install python-dateutil
```
#### cd到项目文件夹下执行python Main.py
